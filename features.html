Features


Dein Name

Den Darkmode kannst du mit einem Klick auf die Sonne, oben rechts, aktivierten bzw. deaktivieren.
Der Zusstand des Darkmodes wir im Local Storage gespeichert.
Er ist somit Seitenübergreifend, zudem nimmt er nach erneutem Besuch der Seite den letzen Zustand wieder an.
Wenn der Zustand noch unbekannt ist (erster Besuch der Seite oder Local Storage gelöscht)
nimmt der Darkmode den Zustand des Systems an.

Wenn du auf eine belibiege Überschrift klickst, erschent ein Pop-up Fesnter mit "Dein Name".
In das Input Feld "Name" kannst du deinen Namen eintragen.
Wenn du auf "Bestätigen" klickst, erscheint eine bestätigungs Nachricht und die Überschrift ändert sich zu

"Moin 'dein Name' - 'Seiten Überschrift'"


Dein Name wird im Local Storage gespeichert und bei jeder Seite wie beschrieben angezeigt.
Wenn du in dem Input Feld "Name" nichts eintippst und auf "Bestätigen" klickst,
wird die standart Seiten Überschrifft angezeigt und es erscheint eine weitere bestätigungs Nachricht.

Wenn du in dem Input Feld "Name" beispeilsweise ein Style Tag eintippsts und auf "Bestätigen" klickst,
erscheint eine Fehlermeldung.
